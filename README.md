# VaporDemo
A demo for Vapor

## Current Environment

**OS**: macOS or Ubuntu

**Database**: MySQL + Redis

**Framework**: [Vapor](https://github.com/vapor/vapor)

## Conifg / secrets

**mysql.json**

```
{
    "hostname": "xx.xx.xx.xx",
    "port": 3306,
    "user": "xxxx",
    "password": "xxxx",
    "database": "xxxx"
}
```

**redis.json**

```
{
    "hostname": "xx.xx.xx.xx",
    "port": 6379,
    "password": "xxxx",
    "database": 0
}
```

**aes.json**

```
{
  "key": "xxxxxxxxxxxxxxxx"
}

```
